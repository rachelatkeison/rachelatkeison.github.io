Selected Projects

These projects reflect the kind of work I’m most excited about: interactive music systems, responsive environments, audio software ideas, and technical experiments shaped by a strong creative point of view.

CloudStage MR Studio

CloudStage MR Studio is an experimental virtual concert environment where performers and listeners interact with a responsive stage, spatial sound, and real-time musical behavior.

I built it to explore the idea that a stage does not have to be passive. It can listen, react, shift with harmony, and become part of the instrument itself.

CloudStageSynthBridge

CloudStageSynthBridge is a JUCE-based audio bridge created as part of the larger CloudStage system. It explores how a Unity-based interactive performance environment can connect to a separate external synth engine.

This project reflects my interest in building music tools that feel imaginative on the surface but still have strong technical structure underneath.
